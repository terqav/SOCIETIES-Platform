Attention!!!

To successfully build, deploy and test this WAB (web application bundle),
follow the few lines of guidelines as below.


1. Build the project as normal using mvn clean install

2. The get the full WebApp experience you should deploy the following plans
	CommsFramework.plan
	ExampleServices.plan
	service-lifecycle.plan 
	societies-webapp.plan
	
3. to get the privasy & trust (and assessment) webapp experience
	privacy and trust plan (privacy-trust/java/privacy-protection)
	context broker plan
	
	Then go to web browser and type http://localhost:8080/societies/login.html to start the web app
